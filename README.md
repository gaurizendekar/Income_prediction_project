# Income Prediction Using Machine Learning

## Overview

This project uses Machine Learning techniques to predict whether an individual's income exceeds a specified threshold based on demographic and employment-related attributes. The project demonstrates the complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and evaluation.

## Features

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Handling missing values and categorical features
* Feature engineering and encoding
* Machine Learning model training
* Model performance evaluation
* Income category prediction

## Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Income Prediction

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/income-prediction-ml.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Open:

```text
incomeproject_final.ipynb
```

The trained machine learning model predicts income categories based on input features and demonstrates practical applications of classification algorithms in predictive analytics.
